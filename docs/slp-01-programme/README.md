## Programme Summary

This 12-week SLP equips departmental heads, team leads, line managers, supervisors, project leads, operations leads, HR practitioners and postgraduate learners to apply generative AI responsibly in workplace and people-management contexts.

The programme moves learners from basic LLM use to responsible work redesign. Learners practise using ChatGPT, Claude and similar tools for common managerial tasks, then build a workplace-ready AI + HRM work redesign playbook as the final portfolio.

## Delivery Assumption

The default design is online or blended distance learning for working adults. It assumes learners have basic digital literacy but not coding or technical AI expertise.
