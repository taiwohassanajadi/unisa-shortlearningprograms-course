## Programme Summary

This 12-week SLP equips HR managers, HR analysts, departmental heads, workforce planners, talent managers, organisational development practitioners, internal auditors, consultants and postgraduate learners to use people analytics and AI-supported analysis responsibly.

The programme treats people analytics as **decision support**, not dashboard decoration. Learners build a workplace-ready **People Analytics Decision Brief** as the final portfolio.

## Delivery Assumption

The default design is online or blended distance learning for working adults. It assumes learners have basic spreadsheet and digital literacy, but not coding, statistics or data science expertise.
