# Case Study and Worksheet Pack

## Case Study 1: Employee Query Triage Agent

### Scenario

An HR shared-services team receives repeated employee questions about leave, benefits, policies and payroll timelines. The team wants an AI agent to triage questions, retrieve policy information and route complex cases to the right HR specialist.

### Learner Task

Prepare an agentic workflow blueprint that includes:

- Current workflow.
- Agent role.
- Data access.
- Approved actions.
- Prohibited actions.
- Escalation paths.
- Audit logs.
- Pilot metrics.

### Discussion Questions

- What questions should the agent answer directly?
- What cases must be escalated to a human?
- What employee data should the agent not access?
- How will incorrect advice be detected and corrected?

## Case Study 2: Recruitment Coordination Agent

### Scenario

A recruitment team wants an AI agent to coordinate interview scheduling, send reminders, gather candidate documents and update recruitment status dashboards.

### Learner Task

Design the readiness assessment and permission map.

### Discussion Questions

- Which actions are low-risk?
- Which actions require human approval?
- What candidate information is sensitive?
- How should the agent handle missing documents or candidate complaints?

## Case Study 3: Learning Administration Agent

### Scenario

An organisation wants an AI agent to recommend learning modules, send reminders, track completion and generate team-level learning progress summaries.

### Learner Task

Create a pilot plan with fairness, privacy and employee-experience safeguards.

### Discussion Questions

- Could recommendations reinforce existing skills gaps?
- Should managers see individual learning behaviour?
- What data should be summarised rather than exposed?
- What metric would show meaningful learning progress?
