# UNISA Short Learning Programmes: AI + HRM Portfolio

This repository contains a research-backed portfolio of five proposed Short Learning Programme concepts for the University of South Africa context.

The seed idea is **AI + HRM for Managers: From LLM Use to Work Redesign**. The portfolio expands it into five standalone SLP concepts for a mixed audience of managers, HR practitioners, researchers, postgraduate learners, and applied policy/strategy professionals.
